# Star Wars Demo Animation
****

## FEATURES:

- Written in assembly ([FASM](https://flatassembler.net/))
- Uses OpenGL to render animation
- Uses linear interpolation
- Camera rotation is a bug :D
- Smooth camera rotation and objects movement
- Looped animation
- 3 star destroyers
- 1 Sid fighter
- 3-color stars
